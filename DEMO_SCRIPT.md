## Recording Tips

1. **Prepare**:
   - Clear localStorage before recording
   - Have Pera Wallet ready with TestNet ALGO
   - Test everything once before recording
   - Close unnecessary tabs

2. **During Recording**:
   - Speak clearly and confidently
   - Show mouse cursor
   - Zoom in on important parts
   - Keep pace steady
   - Smile (if showing face)

3. **What to Show**:
   - Dashboard with campaigns
   - Wallet connection process
   - Campaign creation with image
   - Real transaction in Pera Wallet
   - Updated campaign with contribution
   - My Campaigns page

4. **What to Avoid**:
   - Don't show errors
   - Don't fumble with forms
   - Don't show debug page (unless specifically demoing it)
   - Don't show console logs

5. **Screen Recording Settings**:
   - 1920x1080 resolution
   - 30 FPS
   - Include system audio
   - Use OBS Studio or similar
